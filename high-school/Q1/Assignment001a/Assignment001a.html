As you can see from these examples, you can open hyperlinks in the same browser window or in a new browser window by using the target attribute of the anchor tag.

Things to remember:

  • HTML tags often have attributes (align, target, class, id, href, name, content, type...)
  • Attributes modify how the HTML tags function
  • Attributes should no longer modify the look of the HTML tag
