Tables

Tables are composed of rows and columns. They are used to display data. Remember that:

  • Rows go side to side.
  • Columns go up and down.
  • Table data is contained in cells

The table tag creates a table. Rows are defined using the tr tag and column cells are defined using the td tag.

The following table is composed of four rows and three columns. Each row has three tds of data:
