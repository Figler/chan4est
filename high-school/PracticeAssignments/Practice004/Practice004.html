Hyperlinks (external)

Hyperlinks, are connections to other Web pages. These Web pages can be out on the World Wide Web. For example, Yahoo, Google, and MSN. Hyperlinks are created using the a tag (anchor tag). Look at the source of this page to see how the following three links are created. These are absolute paths because a starting point is given!

  1. Yahoo
  2. Google
  3. Reddit

The a tag uses an attribute (setting) named href. Href stands for hyperlink reference. Set href equal to the desired Web page destination. Surround the word, or image, you want as your hyperlink with the start a tag and end a tag. Make sure to use http:// before the URL (address) of the Web page you want to link to. For example:

http://www.yahoo.com.
